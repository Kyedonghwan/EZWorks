[커뮤니티 메인화면]
- 전체 커뮤니티 최근 글 표기
-> 제목 클릭 시, 커뮤니티 글 상세보기 페이지 이동

[커뮤니티 메뉴 1]
- 가입 커뮤니티 목록 표기
- 개별 커뮤니티 명 클릭 시, 해당 페이지로 이동
- 개별 커뮤니티 설정 클릭 시, 커뮤니티 정보 수정 페이지
- 커뮤니티 만들기 클릭 시, 커뮤니티 개설 페이지

[커뮤니티 개설]
- 커뮤니티 개설 정보(커뮤니티 명, 소개, 공개여부)기입 후 등록

[개별 커뮤니티 메인화면]
- 해당 커뮤니티 최근 글 표기
-> 제목 클릭 시, 커뮤니티 글 상세보기 페이지 이동

[커뮤니티 메뉴2]
- 글쓰기 클릭 시, 해당 커뮤니티 게시판 글쓰기 이동
- 개별 게시판 설정 클릭 시, 게시판 정보 수정 페이지
- 게시판 추가 클릭 시, 게시판 개설 페이지
- 해당 커뮤니티 가입 멤버 목록 표기

[커뮤니티 게시판 개설]
- 게시판 개설 정보
   (게시판 명, 운영자명, 게시판 설명, 유형, 공개여부)기입 후 등록

[커뮤니티 게시판 글쓰기]
- 게시판 선택, 제목, 파일첨부, 내용 기입 후 등록
- summernote 사용
- multipart fileupload 

[커뮤니티 상세보기]
- 커뮤니티 정보(이름, 소개, 운영자, 공개여부 설정) 수정
- 운영자 추가시 모달 사용 -> 가입멤버 중 선택
- 게시판 목록 확인, 수정, 삭제, 추가페이지 이동
- 가입 멤버 현황 확인
